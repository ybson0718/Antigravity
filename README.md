# HackSync (해커톤/대회 운영 플랫폼)

HackSync는 해커톤, 아이디어톤, 코딩 대회 등 다양한 프로젝트 대회를 효율적으로 운영하고 관리할 수 있도록 돕는 올인원 SaaS 플랫폼입니다. 주최자, 심사위원, 참가자 각각의 역할에 최적화된 대시보드를 제공하여 대회 준비부터 심사 결과 집계까지의 모든 프로세스를 자동화합니다.

## 웹페이지
https://antigravity-beige-five.vercel.app/

## 🚀 주요 특징 (Features)

- **운영 리소스 절감**: 참가자 관리, 팀 빌딩, 심사 채점 등을 하나의 플랫폼에서 해결하여 운영 시간을 최대 70% 단축합니다.
- **실시간 데이터 시각화**: Chart.js 기반의 대시보드를 통해 지원자 통계, 팀 매칭 현황, 실시간 심사 결과를 직관적으로 파악할 수 있습니다.
- **Glassmorphism UI**: 모던하고 세련된 다크 테마와 글래스모피즘(Glassmorphism) 디자인 시스템을 적용하여 최상의 사용자 경험을 제공합니다.
- **커스텀 알림 시스템**: 브라우저 기본 알림창을 대체하는 UI/UX 최적화 자체 토스트(Toast) 알림 기능을 구현했습니다.

## 👥 역할별 기능 안내 (Role-Based Dashboards)

HackSync는 사용자의 역할에 따라 동적으로 라우팅되며, 각 역할에 맞는 맞춤형 기능을 제공합니다.

### 1. 주최자 (Organizer)
대회 전반을 기획하고 운영하는 총괄 관리자를 위한 기능입니다.
- **통합 대시보드 (`#dashboard`)**: 전체 지원자 수, 팀 빌딩 현황, 제출율 등 핵심 KPI와 실시간 차트 제공.
- **지원자 관리**: 전체 지원자의 직무, 소속 팀 상태를 확인하고 합격/불합격 여부를 클릭 한 번으로 처리.
- **심사 결과 집계**: 심사위원들이 채점한 내역이 실시간으로 가중치 계산되어 랭킹 보드에 자동 반영.
- **CRM 메시지 발송**: 참가자 그룹(합격자, 미제출자 등)을 타겟팅하여 이메일 및 알림톡 메시지 발송 관리.

### 2. 심사위원 (Judge)
프로젝트 결과물을 평가하고 점수를 부여하는 평가자를 위한 기능입니다.
- **심사위원 대시보드 (`#judge`)**: 자신에게 배정된 평가 대상 팀 목록과 프로젝트 소개, PDF, GitHub 링크 등을 한곳에서 열람.
- **간편한 실시간 채점**: 창의성, 완성도, 발표력 등의 평가 항목을 슬라이더(Slider)로 직관적으로 입력하며 실시간 가중치 총점 확인.
- **채점 내역 관리**: 이미 평가를 마친 팀의 점수 현황을 리스트로 확인.

### 3. 참가자 (Participant)
대회에 참가하여 아이디어를 구체화하고 팀을 꾸리는 사용자를 위한 기능입니다.
- **팀 빌딩 보드 (`#team`)**: 다른 참가자들의 프로필(직무, 스킬, 매칭도 등)을 확인하고 필터링을 통해 최적의 팀원을 탐색 및 초대.
- **팀 생성 및 관리**: 새로운 프로젝트 팀을 생성하고, 현재 멤버를 확인하며 트랙을 설정하는 등 자유로운 팀 관리 가능.

## 🛠 기술 스택 (Tech Stack)

- **Frontend**: HTML5, CSS3 (CSS Variables, Flexbox, CSS Grid), Vanilla JavaScript (ES6+)
- **Routing**: SPA(Single Page Application) Hash 기반 클라이언트 라우팅
- **State Management**: LocalStorage API를 활용한 데이터 영속성 (새로고침 시 데이터 유지)
- **Data Visualization**: Chart.js (Bar, Doughnut, Line charts)
- **Icons**: Lucide Icons

## 💻 실행 방법 (How to Run)

본 프로젝트는 순수 프론트엔드 환경(HTML/CSS/JS)으로 구축되어 별도의 백엔드 서버나 빌드 과정이 필요 없습니다.

1. 프로젝트 폴더 내의 `index.html` 파일을 더블클릭하여 최신 웹 브라우저(Chrome, Edge 등)에서 엽니다.
2. (선택) **VS Code**를 사용하는 경우, `Live Server` 확장을 설치한 후 `index.html`에서 우클릭하여 "Open with Live Server"를 선택하면 더 쾌적하게 테스트할 수 있습니다.
