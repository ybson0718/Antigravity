# HackSync — All-in-One 해커톤·공모전 운영 플랫폼

> **파편화된 대회 운영을 하나의 코어로 동기화하다.**

HackSync는 해커톤, 아이디어톤, 코딩 대회 등 다양한 프로젝트 대회를 효율적으로 운영하고 관리할 수 있도록 돕는 **올인원 SaaS 플랫폼**입니다.  
주최자(Organizer) · 심사위원(Judge) · 참가자(Participant) 각각의 역할에 최적화된 대시보드를 제공하여, 대회 준비부터 심사 결과 집계까지의 모든 프로세스를 자동화합니다.

🔗 **배포 URL**: [https://antigravity-beige-five.vercel.app](https://antigravity-beige-five.vercel.app/)

---

## 🚀 주요 특징

| 특징 | 설명 |
|------|------|
| **운영 리소스 70% 절감** | 참가자 관리, 팀 빌딩, 심사 채점을 하나의 플랫폼에서 해결 |
| **실시간 데이터 시각화** | Chart.js 기반 대시보드로 지원자 통계·팀 매칭 현황·심사 결과를 한눈에 파악 |
| **Glassmorphism UI** | 모던 다크 테마와 글래스모피즘 디자인으로 최상의 사용자 경험 제공 |
| **역할 기반 라우팅** | 로그인 역할(주최자/심사위원/참가자)에 따라 전용 대시보드로 자동 연결 |
| **커스텀 토스트 알림** | 브라우저 기본 알림을 대체하는 UI/UX 최적화 토스트 알림 시스템 |
| **데이터 영속성** | LocalStorage를 활용하여 새로고침 후에도 채점·지원자 상태 유지 |
| **모바일 반응형** | 햄버거 메뉴·사이드바 토글 등 모바일 환경 최적화 |

## 🖥 페이지 구성 및 사용법

### 1. 랜딩 페이지 (`#home`)
웹사이트에 처음 접속하면 보이는 **소개 페이지**입니다.
- 실시간 **카운터 애니메이션**으로 누적 참가자, 진행된 대회 수 등 핵심 지표 표시
- **"단 3단계로 시작하세요"** — 대회 생성 → 참가자 모집 & 팀빌딩 → 심사 & 결과 발표 흐름 안내
- 주최자들의 **후기(Testimonials)** 섹션
- 하단 CTA 배너를 통해 **관리자 뷰 체험** 또는 **참가자 뷰 체험** 바로가기

### 2. 기능 소개 페이지 (`#features`)
HackSync의 **4대 핵심 기능**을 카드형 레이아웃과 상세 딥다이브 섹션으로 소개합니다.
- **동적 대회 생성 & 원클릭 랜딩** — 드래그 앤 드롭 폼 빌더, 자동 랜딩 페이지 생성
- **스마트 팀 빌딩 & 매칭 엔진** — GitHub/Notion 포트폴리오 연동, 스킬 기반 매칭 알고리즘
- **실시간 채점 & 자동 랭킹** — 다중 심사위원 동시 채점, 가중치 기반 자동 총점 계산
- **자동화 CRM** — 합격/불합격 결과를 타겟 그룹에 원클릭 발송
- **기존 방식 vs HackSync 비교 테이블** 제공

### 3. 주최자 대시보드 (`#dashboard`)
대회 전반을 기획·운영하는 **관리자 전용** 화면입니다. 좌측 사이드바로 서브뷰를 전환합니다.

| 서브뷰 | 기능 |
|--------|------|
| **통합 대시보드** | 총 지원자·팀 빌딩 완료율·제출율·채점 진행률 등 KPI 카드 + Bar/Doughnut/Line 차트 + 최근 액티비티 피드 |
| **전체 지원자** | 지원자 목록 테이블 (검색·필터 지원) + 합격/불합격 상태 변경 버튼 |
| **심사 결과 집계** | 실시간 랭킹 보드 — 창의성(35%)·완성도(40%)·발표력(25%) 가중치 자동 계산 |
| **결과 통보 (CRM)** | 수신 대상 세그먼트 선택 → 이메일/알림톡 채널 선택 → 메시지 작성 및 즉시/예약 발송 |

### 4. 심사위원 대시보드 (`#judge`)
프로젝트 결과물을 평가하는 **심사위원 전용** 화면입니다.

| 서브뷰 | 기능 |
|--------|------|
| **배정된 평가 대상** | 자신에게 배정된 팀 카드 목록 — 프로젝트 소개, PDF/GitHub/시연 영상 링크 열람 |
| **채점 완료 내역** | 이미 평가한 팀의 점수 현황 리스트 |

- **채점 모달**: 팀 카드의 "채점하기" 버튼 클릭 → 슬라이더로 창의성/완성도/발표력 점수 입력 → 가중치 적용 총점 실시간 계산 → 제출 시 주최자 랭킹 보드에 즉시 반영

### 5. 참가자 팀빌딩 (`#team`)
대회에 참가하여 팀을 구성하는 **참가자 전용** 화면입니다.

- **내 팀 현황 배너**: 현재 소속 팀 정보, 멤버 아바타 스택, 빈 자리 표시
- **참가자 프로필 카드 그리드**: 이름·직무·스킬 태그·GitHub/Notion 링크·매칭도(%) 표시
- **필터링**: 전체/Frontend/Backend/AI·Data/Design/PM 직무별 필터 + 이름·스킬 키워드 검색
- **팀원 영입**: "영입하기" 버튼 클릭 시 초대 요청 토스트 알림
- **팀 관리 모달**: 팀 이름·트랙·모집 인원·팀 소개 수정, 현재 멤버 확인

### 6. 로그인 / 회원가입 (`#auth`)
- 역할(주최자/심사위원/참가자) 선택 후 이메일·비밀번호로 로그인
- 로그인 시 역할에 맞는 대시보드로 자동 리다이렉트
- 비로그인 상태에서 보호된 페이지 접근 시 로그인 페이지로 안내

## 🛠 구현된 기능 목록

### 프론트엔드 아키텍처
- **SPA (Single Page Application)**: Hash 기반 클라이언트 라우팅 (`#home`, `#features`, `#dashboard`, `#judge`, `#team`, `#auth`)
- **역할 기반 접근 제어**: 로그인 역할에 따른 보호된 라우트 및 네비게이션 동적 변경
- **LocalStorage 상태 관리**: 채점 결과, 지원자 상태, 사용자 정보 등 데이터 영속성 보장

### UI/UX
- **Glassmorphism 디자인 시스템**: CSS Variables 기반 다크 테마 + 글래스 이펙트
- **배경 글로우 이펙트**: 그라데이션 원형 배경으로 깊이감 연출
- **Fade-in-up 애니메이션**: 페이지 진입 시 콘텐츠 순차 등장 효과
- **숫자 카운터 애니메이션**: easeOutExpo 이징 함수 적용
- **Hover Lift 인터랙션**: 카드 호버 시 상승 효과
- **커스텀 토스트 알림**: success/error/info 타입별 아이콘 및 스타일 분기
- **반응형 레이아웃**: 모바일 햄버거 메뉴, 사이드바 토글, CSS Grid/Flexbox 반응형
- **스크롤 연동 네비바**: 스크롤 시 배경 투명도 동적 변경

### 데이터 시각화 (Chart.js)
- **Bar Chart**: 직무별 지원자 분포 (그라데이션 바)
- **Doughnut Chart**: 포지션 비율 (범례 하단 배치)
- **Line Chart**: 일별 지원자 추이 (영역 채우기 + 포인트 스타일)

### 인터랙티브 기능
- **지원자 관리**: 검색 필터링 + 상태(심사 대기/합격/불합격) 필터 + 원클릭 상태 변경
- **심사 채점 시스템**: 슬라이더 입력 → 가중치 실시간 계산 → 주최자 랭킹 보드 자동 반영
- **팀빌딩**: 참가자 검색·직무별 필터 + 매칭도 표시 + 팀 초대 + 팀 생성/관리 모달
- **CRM 메시지 발송**: 수신 대상 세그먼트 선택 + 채널(이메일/알림톡) 토글 + 메시지 작성·발송
- **인증 시스템**: 로그인/회원가입 폼 전환 + Mock 계정 인증 + 역할별 리다이렉트

## 💻 실행 방법

본 프로젝트는 순수 프론트엔드 환경(HTML/CSS/JS)으로 구축되어 **별도의 백엔드 서버나 빌드 과정이 필요 없습니다.**

1. 프로젝트 폴더 내의 `index.html` 파일을 더블클릭하여 최신 웹 브라우저(Chrome, Edge 등)에서 엽니다.
2. (선택) **VS Code**를 사용하는 경우, `Live Server` 확장을 설치한 후 `index.html`에서 우클릭 → **"Open with Live Server"** 를 선택하면 더 쾌적하게 테스트할 수 있습니다.

로컬 스토리지에 기본 내장된 아래 계정들을 통해 역할별 기능을 테스트해 보실 수 있습니다.
